กิจกรรม 2.3 การค้นหาข้อมูลแบบทวิภาค

💡 ขั้นตอนวิธี : การค้นหาข้อมูลแบบทวิภาค (Binary Search Algorithm)

ข้อมูลเข้า: รายการ L ที่มีข้อมูลการเรียงลำดับจากน้อยไปหามาก และข้อมูลที่ต้องการค้นหา (target)
ข้อมูลออก: ค่าดัชนี i ในรายการ L ที่บอกตำแหน่งของ target (หากไม่พบข้อมูลจะคืนค่าบรรลุเงื่อนไขสิ้นสุด)
ขั้นตอนวิธี:
  1. ให้ n แทนจำนวนข้อมูลในรายการ L
  2. ให้ left = 1
  3. ให้ right = n
  4. ทำซ้ำในขณะที่ left = right:
    • ให้ mid มีค่า (left + right) / 2 ปัดเศษทิ้ง
    • ให้ x แทนข้อมูลในรายการ L ณ ตำแหน่งดัชนี mid
    • ถ้า x เท่ากับ target ให้ คืนค่าดัชนี mid$และจบการทำงาน
    • ถ้า x มากกว่า target ให้ right มีค่า mid - 1
    • ถ้า x น้อยกว่า target ให้ left มีค่า mid + 1
รายการข้อมูลเริ่มต้นก่อนเรียงลำดับ: 20, 16, 8, 7, 6, 10, 11, 12, 14, 22, 15, 17, 28, 29, 30, 4, 2, 21
